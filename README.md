# GADC WDI20 Project 3 Back End: Netflix and Chill

## Technologies

- Express
- MongoDB
- Mongoose
- Body-Parser
- Cors

## Routes

`/movies/`

- [x] GET: returns all movies
- [ ] POST: add a new movie to 'movies' collection in the database

`/movies/:id`

- [ ] PUT: update a specific movie
- [ ] DELETE: delete a specific movie
